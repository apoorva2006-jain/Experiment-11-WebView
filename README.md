# Experiment-11-WebView 
Aim:
To design and develop an Android application that uses a Android Studio WebView component to display a web page inside the app.

Algorithm:
Start
Open Android Studio
Create a new Android project
Select Empty Activity
Open the layout file (activity_main.xml)
Add a WebView component in the layout
Assign an ID to the WebView (e.g., webView)
Open MainActivity.java (or .kt file)
Import the WebView package
Create a WebView object using findViewById()
Enable JavaScript in WebView settings
Set a WebViewClient to handle web page loading inside the app
Load a URL using loadUrl() method (e.g., google.com)
Add Internet permission in AndroidManifest.xml
Run the application on emulator or device
Verify that the web page loads inside the app
Stop

Result:
The Android application successfully displays a web page using WebView inside the app.
