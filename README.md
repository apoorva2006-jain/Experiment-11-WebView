# Experiment-11-WebView 

Aim

To create an Android application that displays a web page inside the app using Android Studio WebView.

Algorithm

Start Android Studio.

Create a new Android project.

Select Empty Activity template.

Open activity_main.xml file.

Add a WebView control in the layout.

Open MainActivity.java file.

Initialize WebView using findViewById().

Enable JavaScript and set WebViewClient.

Load a URL using loadUrl() method.

Run the application on emulator or mobile device and verify the web page is displayed.
